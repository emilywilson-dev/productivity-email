# productivity-email

Responsive email template designed for cross-client compatibility

## Features
- Fully responsive layout
- Tested in Gmail, Outlook, and Apple Mail
- Dark mode support
- VML buttons for Outlook
- Safe background images with VML fallback
- Modular, clean code
